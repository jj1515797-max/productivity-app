# 안돈 생산관리 시스템

내포장 / 외포장 / 잔여량 실시간 관리 웹앱.

## 기술 스택
- React + Vite + TypeScript + Tailwind CSS
- Firebase (Firestore + Auth)
- GitHub Pages (호스팅)

## 로컬 실행
```bash
npm install
npm run dev
```

## 배포
`main` 브랜치에 푸시하면 GitHub Actions가 자동으로 GitHub Pages에 배포합니다.

배포 URL: `https://jj1515797-max.github.io/productivity-app/`
